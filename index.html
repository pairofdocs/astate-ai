AI Opportunities for Allstate Investment Group

Leveraging AI to unlock value from structured and unstructured data

Why AI now?

  • Explosion of data (structured & unstructured)
  • Advances in AI models (LLMs, NLP, computer vision)
  • Competitive advantage through smarter insights

AI Tech

  • State of the art tool
  • AI as a generator. Text, code, ideas, emails, visuals
  • AI as a workflow assistant. Automate processes
  • AI as a model enhancer. Predict return, recommend, portfolio rebalance

Current Data Challenges at Allstate

  • Massive volumes of unstructured data
  • Fragmented data systems
  • Time-consuming manual processes

Structured Data Use Cases

  • Predictive analytics for investment performance
  • Automated portfolio risk scoring

Unstructured Data Use Cases

  • NLP to extract insights from analyst reports & earnings calls
  • Sentiment analysis from news & social media
  • Document classification, summarization, transform to structured data

How AI Helps

  • Reduces time spent on data prep & analysis
  • Improves forecasting accuracy
  • Identifies patterns not visible to humans
  • Enables real-time decision-making

Getting Started

  • Prioritize high-value use cases
  • Use cloud platforms and pre-trained models
  • Build cross-functional teams (data + domain experts)
  • Start small, iterate fast

Final Thoughts

  • AI is not a silver bullet—but a powerful tool & enabler
  • Start with data you already have
  • Aim for solid foundation and wise use
  • Build momentum
